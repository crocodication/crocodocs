Preparations

1. Install Node.js

Install Node.js, this package is running under Node.js environment, the most recommended way to install it is via NVM

Get Node.js via NVM

2. Install Crocodocs Globally

Run "npm i crocodocs -g" on command prompt

3. Get To Know How It Works

Mostly Crocodocs works by commenting your code with specific comment pattern and then when you generate, it will automatically lists all your script by itself

Find out more how it works

4. Generate

Run "crocodocs create"
